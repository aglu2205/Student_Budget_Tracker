# Student Budget Tracker

A simple Java-based command-line application that allows users to track income and expenses.

## Features
- Add income
- Add expenses
- View current balance
- Interactive CLI menu
- Object-oriented design

## Tech Stack
- Java
- OOP principles
- Git & GitHub

## Project Structure
src/
- Main.java
- BudgetTracker.java

## How to Run
1. Compile the files
2. Run Main.java
3. Follow menu prompts
